# 💞 E2Entim

**Secure messenger for couples who value privacy**

> *For nerds in love, who don't trust Big Tech with their nudes*

## Что такое E2Entim?

Клиент-серверное приложение для безопасного обмена:
- Сообщениями
- Фотографиями 
- Видеороликами

**P.S. Моногамия онли!** Приложение создавалось для парочек, а не для гаремов или изменщиков. 
Если вы ищете Tinder, но с E2E и приватностью - то вы лох, хахаххаха

## Как запустить это чудо?

Честно: я сам только закончил проверять код на уязвимости. Сейчас буду покупать VPS, чтобы начать тестить бета-версию.

Есть .NET сервер и React-Native клиент. Их нужно собрать и получить чудо новогоднее.

### Быстрый старт:

**Server** (можно вроде сделать докер, но я пока не делал, короче сами тыкайте солюшен)

    cd E2EEIntimServ
    dotnet restore
    dotnet run

**Client**

    cd E2EEntim_Client
    npm install
    npm run android

или для iOS (не тестировал):

    npm run ios

## Зависимости

### Клиент (React Native)

- React Native 0.85.3 - Мобильный фреймворк
- TypeScript - Типизация
- @microsoft/signalr - WebSocket (SignalR)
- tweetnacl - Криптография E2EE
- tweetnacl-util - Утилиты для криптографии
- react-native-mmkv - Локальное хранилище
- react-native-keychain - Безопасное хранилище ключей
- react-native-async-storage - Хранилище данных
- react-native-image-picker - Выбор фото/видео
- react-native-video - Воспроизведение видео
- react-native-fs - Файловая система
- react-native-compressor - Сжатие видео
- @react-native-camera-roll/camera-roll - Сохранение в галерею
- react-native-safe-area-context - Адаптация под Safe Area
- react-native-screens - Оптимизация навигации
- react-native-gesture-handler - Обработка жестов
- react-native-reanimated - Анимации
- react-native-paper - UI компоненты
- react-native-vector-icons - Иконки
- @react-navigation/native - Навигация
- @react-navigation/native-stack - Стековая навигация

### Сервер (.NET)

- .NET 10.0 - Фреймворк
- ASP.NET Core SignalR - WebSocket
- Entity Framework Core - ORM
- SQLite - База данных

**Важно:** Нюгет ругается на уязвимость SQLite. Поэтому есть вероятность, что он тут не надолго. Если знаете, как починить - сделайте форк.

## Безопасность

Что мы имеем:

- End-to-End Encryption (E2EE)
- Приватные ключи никогда не покидают устройство
- Шифрование видео по частям
- Защищённое локальное хранилище

Что в разработке:

- Экран блокировки
- Кружочки
- Решение бага, что на андроиде клава забавна бегает
- Реакции

... и другие приколюхи!
